## Project Overview

ShoeShop is an ecommerce app for buying of shoes, its built with flutter, it has a beautiful and responsive user interface which gives detailed information about a particular item available for sale. The author of this project is [TheAlphamerc](https://github.com/TheAlphamerc) ![GitHub release (latest by date)](https://img.shields.io/github/v/release/TheAlphamerc/flutter_ecommerce_app)
